最新版本更新内容 V7.8.0.62

  • 1. 新增 梦之回廊模拟器 网页及入口
  • 注意：本次更新的页面为 GitHub 独占，规避可能存在的风险
    • 1.1: 基于 官方概率公示 与 推理保底算法 的碎片抽奖模拟（请勿代入现实，仅为概率模拟）
    • 1.2: 完整实现 25 碎片集齐系统，含 300 抽硬保底与 250 抽软保底机制
    • 1.3: 支持 三种抽奖模式：
      • 单抽 – 每次抽取 1 次
      • 十连抽 – 每次抽取 10 次
      • 三十连抽 – 每次抽取 30 次
    • 1.4: 实时显示 花费统计：
      • 累计抽取次数
      • 累计花费金额（每抽 = 3 元）
      • 当前已拥有碎片数量 / 25
    • 1.5: 特色功能：
      • 完整 概率公示表，数据源自官方公布（仅供参考，实际以游戏为准）
      • 响应式设计，完美适配桌面和移动设备
      • 碎片收集进度可视化 – 25 宫格实时点亮
      • 抽取日志倒序置顶，最新记录一目了然
    • 1.6: 该页面可单独下载使用，脱离联机版运行时仅页顶页尾 CSS 失效
    • 1.7: 重要提醒：本模拟器所有数据均为虚拟生成，与《三国杀》官方实际活动无关，请勿据此进行任何消费决策！
  • 2. 新增 开袋子模拟器 网页以及入口
    • 2.1: 基于 权重随机算法 的纯概率抽奖模拟（请勿代入现实，仅为概率模拟）
    • 2.2: 完整实现 开袋子 抽奖系统，无保底机制
    • 2.3: 支持 三种抽奖模式：
      • 单抽 – 每次抽取 1 个袋子
      • 10 连抽 – 每次抽取 10 个袋子
      • 50 连抽 – 每次抽取 50 个袋子
    • 2.4: 实时显示 花费统计：
      • 累计抽取次数
      • 累计花费银币数量（每个袋子 = 100 银币）
      • 累计消耗史诗宝珠数量（每个袋子 = 0.01 个史诗宝珠）
    • 2.5: 特色功能：
      • 抽到 "当期史诗武将*1" 时 弹窗提示（可开关）
      • 完整的 概率公示表，数据基于指定权重计算（仅供参考，实际以游戏为准）
      • 响应式设计，完美适配桌面和移动设备
      • 支持 回车键和空格键快捷 50 抽，可长按连续抽取
    • 2.6: 该页面可单独下载使用，脱离联机版运行时仅页顶页尾 CSS 失效
    • 2.7: 重要提醒：本模拟器所有数据均为虚拟生成，与任何游戏官方实际活动无关，请勿据此进行任何消费决策！
  • 3. 核心算法特性
    • 3.1: 梦之回廊：基于 官方碎片掉落概率，1~22 片按固定表递减；23~25 片进入 soft-pity，240 抽后概率线性飙升，271 拉起必出，300 抽硬保底必齐
    • 3.2: 开袋子：基于 权重随机算法，严格按照指定权重进行概率分布，无保底机制，每次抽取均为独立随机事件
    • 3.3: 智能花费计算，实时更新银币、史诗宝珠或人民币消耗
  • 4. 概率设置
    • 4.1: 梦之回廊：第 1 片 100% → 第 22 片 1.95% 逐级递减；第 23 片起进入 soft-pity，240 抽后基础概率 5% 起跳
    • 4.2: 开袋子（基于指定权重）：严格按照官方给出的概率执行
  • 5.将“开盒子模拟器”、“开袋子模拟器”、“梦之回廊模拟器”页面在新版主页里归类于“github限定”分类
